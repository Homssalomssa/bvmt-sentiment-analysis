# bvmt-sentiment-analysis

Repository for the **BVMT Trading Assistant** (IHEC-CODELAB 2.0 Hackathon).

## Sentiment Analysis module

The **Sentiment Analysis** module (Person 3) lives in the **[sentiment](sentiment/)** folder.

- **Setup & run:** open the [sentiment](sentiment/) folder and follow its [README](sentiment/README.md).
- **API:** run the server from `sentiment/`; it listens on `http://localhost:8001`.

```cmd
cd sentiment
setup.bat
python api.py
```

See **[sentiment/README.md](sentiment/README.md)** for full documentation, API details, and team integration.
